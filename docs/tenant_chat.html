Chat

For å kunne sende meldinger og andre vedlegg til forskjellige medlemmer av dine eiendommer, kan man opprette en chat med personen(e). Det er kun medlemmene av en chat som kan se meldingene som blir sendt og chatter kan bestå av både to og flere personer.

Opprette nye chatter

For å kunne sende meldinger i en chat, er man først nødt til å opprette en chat.

Mobil
Web
  1. Åpne Eiendomsappen og trykk på -ikonet nederst på skjermen
  2. Trykk på -ikonet øverst til høyre på skjermen
  3. Velg personene som skal være medlemmer av chatten
  4. Trykk på Ferdig-knappen øverst til høyre på skjermen
  5. Skriv den første meldingen i feltet på bunnen av skjermen
  6. Trykk på SEND-knappen til høyre for meldingen du skrev inn

Det er ikke mulig å opprette nye chatter fra web enda
